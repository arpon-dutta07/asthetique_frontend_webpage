# AESTHETIQUE — Timeless Luxury Interior Design Studio

AESTHETIQUE is a premium, high-end single-page portfolio landing page crafted for a luxury interior design studio. Designed with an editorial aesthetic, it showcases private residences, creative design journeys, bento section modules, and dynamic interactive interfaces.

## ✨ Key Features

- **Responsive Bento Grid Layout**: A sleek, structured bento container that highlights key projects, branding, and navigation options natively.
- **Ambient Lighting overlays**: Integrated Light & Dark theme toggle featuring custom CSS variables and view transitions that slide a smooth curtain effect across the screen.
- **Smart Autoplay Video Banner**: A parallax loop video hero section. Leverages a fail-safe unmuting script to navigate strict browser Autoplay Policies, unmuting on the first gesture and muting automatically when scrolled out of view.
- **Interactive 3D Carousel Showcase**: Vertical sliding showcase linking interactive categories with 3D stacked image representations.
- **Editorial Accordion Gallery**: Expandable custom slider displaying high-resolution project captures and layout specs.
- **Live World Clocks**: Synchronized real-time clocks displaying Soho (NYC), London (UK), and Kyoto (JP) local studio times.
- **Floating Mobile Hamburger**: Sticky navigation shortcut positioned relative to viewport bounds on mobile, enabling easy access from anywhere on tall pages.
- **Custom Brand Monogram**: Standardized custom luxury vector monogram "A" implemented across the title bar, hero logo, side drawer, and overlay details.

## 🚀 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (version 18+ recommended).

### Installation

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/arpon-dutta07/asthetique_frontend_webpage.git
   ```
2. Navigate to the project root directory:
   ```bash
   cd asthetique_frontend_webpage
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally

To launch the local development server with Hot Module Replacement (HMR):
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:5173/` to view the page.

### Building for Production

To compile and bundle optimized static assets:
```bash
npm run build
```
The output files will be written to the `dist/` directory, ready to be deployed to your hosting provider.
